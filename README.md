# AI 탐색기 🚀

급등 조짐 종목 실시간 탐지 시스템 (한국투자증권 KIS API 연동)

## 배포 방법 (Railway - 무료)

### 1단계: GitHub에 올리기
1. https://github.com 에서 새 저장소(repository) 생성
2. 이 폴더의 파일들을 전부 업로드

### 2단계: Railway 연결
1. https://railway.app 접속 → Google/GitHub 로그인
2. **New Project** → **Deploy from GitHub repo**
3. 방금 만든 저장소 선택
4. 자동 배포 시작 (2~3분)
5. **Settings → Domains → Generate Domain** 클릭
6. 생성된 URL을 북마크에 저장

### 사용 방법
- 생성된 URL을 모바일 브라우저에서 열기
- KIS APP Key + APP Secret 입력 후 연결
- (선택) 설정 탭에 Claude API Key 입력 시 뉴스 분석 가능

## 주요 기능
- 📡 급등 조짐 탐지 (거래량 급증 + 공구리 패턴 + 이평 돌파 스코어링)
- 🚀 실시간 급등 종목
- 📰 Claude AI 뉴스 분석 + 종목 연결
- ⏱ 자동 갱신 (설정 가능)
- 📱 모바일 최적화
